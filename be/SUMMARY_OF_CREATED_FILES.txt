================================================================================
TROUBLESHOOTING
================================================================================

Problem:  "Cannot connect to server"
Solution: Start server first: python manage.py runserver

Problem:  "ModuleNotFoundError"
Solution: cd d:\datn\be (run from correct directory)

Problem:  "IntegrityError: duplicate key"
Solution: python manage.py flush (to clear database)
          python manage.py migrate (re-create tables)
          python manage.py shell < load_sample_data.py (reload)

Problem:  "Database is empty"
Solution: Check: python manage.py shell < load_sample_data.py output
          Verify: Database record counts (see verification checklist)

Problem:  "Test hangs or timeout"
Solution: Increase server timeout or restart

Problem:  "API returns 401 Unauthorized"
Solution: Get new token: POST /api/users/auth/login/

For more help:
  → See DATA_LOADING_GUIDE.md (detailed guide)
  → See TESTING_AND_DATA_SUMMARY.md (quick reference)
  → Check Django logs: python manage.py runserver output
